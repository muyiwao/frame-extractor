# 🎬 Frame Extractor Web App

This is a free web app that extracts frames from videos using Python and FFmpeg.

## 🚀 Features

- Upload MP4 videos
- Extract frames in:
  - Batch mode (every X seconds)
  - Single frame (timestamp)
- Choose resolution:
  - 1080p
  - 2K
  - 4K
- Download frames as ZIP

## 🛠️ Tech Stack

- Streamlit
- FFmpeg
- Python

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
